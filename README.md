# Bank Management System - CodeIntern C++ Internship Project

## Description
This project simulates a real-world banking system. Users can:
- Open a new account
- Deposit funds
- Withdraw funds (with minimum balance enforcement)
- Check balance
- Display account details

The project demonstrates:
- Object-Oriented Programming (OOP)
- Menu-driven system
- Banking rules and validations

## Technologies
- C++
- File handling (optional in online compilers)
- In-memory data storage for Programiz compatibility

## Note
This version is fully compatible with online compilers like Programiz. File handling can be enabled when run on local systems.
